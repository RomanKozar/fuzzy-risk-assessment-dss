## 📋 Опис

Програмна технологія підтримки прийняття рішень (ППР) для оцінювання рівня ризику функціонування складних систем на основі **нечіткої моделі** з гібридним аналізом даних.

Система реалізує 4-кроковий алгоритм:

1. **Фазифікація** вхідних гібридних даних (квадратичний S-сплайн)
2. **Агрегування** фазифікованих даних в межах складної системи
3. **Врахування важливості** об'єктів дослідження (вагові коефіцієнти)
4. **Дефазифікація** — обчислення ризик-орієнтованої оцінки R(S)

## 🧮 Математична модель

### Загальна модель оцінювання

```
Δ(t; δ) → τ(R(S); Y)
```

### Крок 1 — Фазифікація (квадратичний S-сплайн)

```
χᵢⱼ = √(δᵢⱼ/2) · (b−a) + a,           якщо 0 ≤ δᵢⱼ ≤ 0.5
χᵢⱼ = b − √((1−δᵢⱼ)/2) · (b−a),       якщо 0.5 < δᵢⱼ ≤ 1

θᵢⱼ = χᵢⱼ / a(k+1)
```

### Крок 2 — Агрегування

```
φᵢ = (1/mᵢ) · Σ θᵢⱼ
```

### Крок 3 — Нормовані вагові коефіцієнти

```
vᵢ = wᵢ / Σwᵢ
```

### Крок 4 — Дефазифікація

```
R₃(S) = Σ vᵢ · (1 − φᵢ)
```

## 📌 Приклад вхідних даних (авіапідприємство)

Терм-множина: `T = {T1 — незначний, T2 — низький, T3 — середній, T4 — високий, T5 — граничний}`

Числова шкала: `[0; 100]`, де T1→[0;20], T2→[20;40], T3→[40;60], T4→[60;80], T5→[80;100]

| Фактор               | Терм | δ   | w   |
| -------------------- | ---- | --- | --- |
| R1 — Перерви послуг  | T5   | 0.9 | 10  |
| R2 — Конкуренція     | T3   | 0.7 | 8   |
| R3 — Стагнація ринку | T2   | 0.5 | 7   |
| ...                  | ...  | ... | ... |

**Результат:** `R₃(S) = 0.613` → рівень керованості **нижче середнього**

## 📖 Як користуватися

1. Відкрийте `index.html` у браузері
2. На вкладці **Конфігурація** — перегляньте терм-множину та вкажіть назву системи
3. На вкладці **Об'єкти дослідження** — додайте об'єкти або натисніть **«Завантажити приклад з ЛР»**
4. Для кожного об'єкта вкажіть: терм T, значення δ (слайдер), вагу w
5. Натисніть **«⚡ Розрахувати R(S)»**
6. Перейдіть на вкладку **Результат** — отримайте оцінку з поясненням

## 📚 Теоретична база

Лабораторна робота №3 — «Програмна технологія підтримки прийняття рішень для оцінювання рівня ризику функціонування складних систем».

Метод базується на нечіткій логіці та гібридному аналізі даних:

- Лінгвістичні оцінки експертів (терм-множина T)
- Кількісні оцінки достовірності δ ∈ [0; 1]
- Вагові коефіцієнти важливості w ∈ [1; 10]
