📐 Математична модель

Нечітка модель оцінювання ризику функціонування системи:

Δ(t; δ) → τ(R(S); Y)

Крок 1. Фазифікація — функція належності (квадратичний S-сплайн):

χᵢⱼ = √(δᵢⱼ/2)·(b−a) + a, якщо 0 ≤ δᵢⱼ ≤ 0.5
χᵢⱼ = b − √((1−δᵢⱼ)/2)·(b−a), якщо 0.5 < δᵢⱼ ≤ 1

θᵢⱼ = χᵢⱼ / a(k+1)

Крок 2. Агрегування в межах об'єктів дослідження:

φᵢ = (1/mᵢ) · Σ θᵢⱼ, j=1..mᵢ

Крок 3. Нормовані вагові коефіцієнти:

vᵢ = wᵢ / Σwᵢ

Крок 4. Дефазифікація — ризик-орієнтована оцінка:

R₃(S) = Σ vᵢ · (1 − φᵢ), i=1..n
R ∈ (0.8; 1] Низький ризик
R ∈ (0.6; 0.8] Нижче середнього
R ∈ (0.4; 0.6] Середній
R ∈ (0.2; 0.4] Вище середнього
R ∈ [0; 0.2] Високий ризик
